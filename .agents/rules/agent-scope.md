---
title: Escopo de Agentes por Projeto
---

Descrição:

Durante o desenvolvimento de uma feature, utilize apenas os agentes estritamente necessários para o projeto ou subprojeto alvo.

Regras de aplicação:
- Identifique o conjunto de agentes suportados pelo projeto (manifestos locais, documentação ou PRD).
- Evite invocar agentes globais ou de outros projetos a menos que haja uma necessidade explícita e documentada.
- Registre no log da feature quais agentes foram usados e por qual motivo.
