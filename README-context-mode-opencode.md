# Context Mode

## Instalação

Para configurar o arquivo `AGENTS.md` para o opencode, execute:

```bash
cp /home/clh/.nvm/versions/node/v22.13.1/lib/node_modules/@anthropic-ai/context-mode/configs/antigravity/AGENTS.md ./AGENTS.md
```

Este comando copia o arquivo de configuração padrão do context-mode para o diretório atual.
